# 🐘 Full-Stack PHP Management System

Este é um projeto de final de curso/módulo que consiste em um sistema de gestão completo desenvolvido em **PHP** e **MySQL**. O foco principal foi a criação de uma aplicação funcional que gerencia o ciclo de vida da informação (CRUD) com foco em segurança e organização de código.

## 🚀 Principais Implementações
- **Painel Administrativo:** Interface desenvolvida para facilitar a gestão de registros.
- **Integração com MySQL:** Estruturação de banco de dados relacional com tabelas vinculadas.
- **Lógica de Back-end:** Processamento de formulários, filtros de busca e manipulação de variáveis globais do PHP.
- **Segurança Básica:** Implementação de verificações de login e proteção de rotas.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** PHP 7/8.
- **Banco de Dados:** MySQL / MariaDB.
- **Front-end:** HTML5, CSS3 e JavaScript para interações básicas.
- **Servidor:** Ambiente Apache (XAMPP/WAMP).

## 🧠 Arquitetura do Sistema
O projeto segue uma estrutura organizada para facilitar a manutenção:
1. **Config:** Arquivos de parametrização e conexão com o banco.
2. **Assets:** Estilização (CSS) e scripts de front-end.
3. **Core/Functions:** Scripts PHP responsáveis pela lógica de negócio e queries SQL.

## 📥 Como Rodar o Projeto
1. Clone este repositório na pasta `htdocs` (XAMPP) ou equivalente.
2. Importe o arquivo `.sql` (se disponível) para o seu gerenciador de banco de dados (phpMyAdmin).
3. Ajuste as credenciais de acesso no arquivo de conexão.
4. Acesse via `localhost/ProjetoFinalPHP` no seu navegador.
